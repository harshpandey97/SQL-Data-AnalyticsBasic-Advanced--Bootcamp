## 📌 Overview

This repository documents my end-to-end SQL learning journey — from writing my first `SELECT` statement to designing optimized queries for large, messy, real-world datasets. It's built to double as both a **learning log** and a **portfolio artifact** for Data Analyst / BI / AI Analyst roles.

> 🎯 **Goal:** Build interview-ready SQL fluency and showcase it through practical, recruiter-facing projects — not just syntax practice.

## 🗺️ Learning Roadmap

<details open>
<summary><b>🟢 Phase 1 — SQL Foundations</b></summary>

- [ ] Database & RDBMS concepts, SQL vs NoSQL
- [ ] `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`
- [ ] Filtering: `IN`, `BETWEEN`, `LIKE`, `IS NULL`
- [ ] Data types & schema design basics
- [ ] `INSERT`, `UPDATE`, `DELETE`, `CREATE TABLE`

</details>

<details>
<summary><b>🟡 Phase 2 — Intermediate: Aggregation & Joins</b></summary>

- [ ] `GROUP BY`, `HAVING`, aggregate functions
- [ ] `INNER`, `LEFT`, `RIGHT`, `FULL OUTER` joins
- [ ] Self joins & cross joins
- [ ] Subqueries (scalar, correlated, nested)
- [ ] Set operations: `UNION`, `INTERSECT`, `EXCEPT`

</details>

<details>
<summary><b>🟠 Phase 3 — Advanced SQL</b></summary>

- [ ] Window functions: `ROW_NUMBER`, `RANK`, `LAG`/`LEAD`
- [ ] CTEs & recursive CTEs
- [ ] Views, indexes & query execution plans
- [ ] Stored procedures, functions & triggers
- [ ] Query performance tuning & optimization
- [ ] Transactions & `ACID` properties

</details>

<details>
<summary><b>🔴 Phase 4 — Analytics & Real-World Application</b></summary>

- [ ] Cohort analysis & retention queries
- [ ] Time-series & rolling aggregates
- [ ] Data cleaning with SQL (dedup, normalization)
- [ ] Building analyst-ready datasets for Power BI
- [ ] End-to-end business case studies

</details>
